Python Introduction

Download Python

Before we can start you first need to have a working version of Python on your system. For most linux users you will already have some version of python installed.

For Windows users please follow the below steps:

  1. Create a folder on your machine where you will be doing your python development. You can do this anywhere you want. Call this folder python.
  2. Open the following link in your browser.
https://www.python.org/downloads/windows/
  1. To download python click on Windows x86-64 embeddable zip file for the latest version under Stable Releases
  2. Once you have downloaded the file copy it to the folder you created in the first step.
  3. Now extract the zipped file in your folder.
  4. Once this is done you should have a folder that looks similar to this one:

Create Hello World Python File

Now that you have downloaded Python we need to test that it works. To do this we will create a Hello World Python program.

  1. Create a new folder in your Python folder called code

  2. In this folder create a new blank text file called helloWorl.py. Make sure the file extension is “.py” and not “.txt”. a. For help creating a blank file see this link: https://www.groovypost.com/howto/quickly-create-new-blank-text-file-windows-mac-linux/ b. For help showing file extensions see this link: https://helpx.adobe.com/x-productkb/global/show-hidden-files-folders-extensions.html

  3. After this is done you should have a file that looks like this one .

Download Visual Studio Code

Before we can start writing code we need an IDE(integrated development environment). We will be using Visual Studio Code as our IDE with some extra plugins.

A big part of being a software developer is looking up how to do certain things online.

To get some practice with this I’m not going to provide detailed steps on how to install Visual Studio Code. I’m only going to provide you with the expected result and you will need to figure out how to do it.

Expected Results:

  1. VS code installed and the helloWorld.py file open

  2. Python extension installed

  3. Terminal extension installed

Good software developers are lazy. They always find easier ways of doing things. For example asking people to figure out how to install visual code on their own so you don’t have to document it.
